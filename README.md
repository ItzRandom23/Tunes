## Installation

To install and run the bot locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ItzRandom23/Tunes.git
   ```

2. **Navigate to the repository directory:**
   ```bash
   cd Tunes
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the bot:**
   ```bash
   npm start
   ```
